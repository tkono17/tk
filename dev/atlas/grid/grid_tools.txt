DQ2 client tools

- dq2_ls

  List the file names of a given dataset

  Examples:

> dq2_ls -g trig1_misal1_csc11.005105.PythiaWmunu.recon.AOD.v12000601
  List all files in this dataset.

> dq2_ls -f -p -s LYON trig1_misal1_csc11.005105.PythiaWmunu.recon.AOD.v12000601
  List all files (physica file name) in this dataset available in the 
  specified site.
